# Glossary

| Term | Definition |
|------|------------|
| ADV | Average Daily Volume (20-day) used for sizing caps |
| Alpaca | Broker API (paper/live) for order routing |
| Alpha Vantage | Market data vendor (HTTP + MCP) for intraday/EOD bars |
| DAL | Data Abstraction Layer; vendor-agnostic fetch + normalization layer |
| HMM | Hidden Markov Model used for regime detection |
| Kelly fraction (`f*`) | Risk sizing output derived from probabilistic expectancy |
| MCP | Managed Capability Provider integration (remote execution) |
| OTEL | OpenTelemetry instrumentation for traces, metrics, logs |
| SignalFrame | Normalized structure containing bars + probabilistic features |
| Streamlit UI | Planned thin client for console/dashboard |

See also: [Sources Reference](./sources.md), [Risk Policy](./risk-policy.md).
