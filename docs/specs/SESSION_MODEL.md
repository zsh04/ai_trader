# Session Model & Analytics

Sessions (PT): PRE 04:00–09:30; REG-AM 09:30–11:30; REG-MID 11:30–14:00; REG-PM 14:00–16:00; AFT 16:00–20:00.

**Tagging:** every signal/order/fill/journal entry is stamped with session.

**Per-session KPIs:** PnL, hit rate, expectancy, MAE/MFE, average spread at entry, slippage, holding time, confidence vs outcome correlation, rule hits.

**Dashboards:** session switcher filters all charts and tables.
