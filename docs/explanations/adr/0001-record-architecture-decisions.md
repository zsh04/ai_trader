# 0001 – Record architecture decisions

Status: Accepted
Date: 2025-11-06
Deciders: Zishan Malik

## Context

We will track significant technical decisions using MADR (<https://github.com/adr/madr>).

## Decision

Adopt ADRs stored under `docs/explanations/adr/`, numbered sequentially.

## Consequences

- Discoverable decision history
- Easier onboarding and audits
